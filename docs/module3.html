Key NCBI Resources

BioProject, BioSample, and SRA

BioProject: Umbrella for related datasets (e.g. surveillance studies)

BioSample: Metadata for a single sample (organism, collection site, etc.) and link to Bioproject and other data

SRA (Sequence Read Archive): Raw fastq reads

Refseq - non-redundant sequence data

Genbank - all deposited assemblies

Retrieving Data from NCBI via Command Line

ncbi-genome-download

Download reference genomes by organism or taxon ID

Filtering by assembly level, refseq/genbank, etc.

Example: ncbi-genome-download bacteria –genus Salmonella –output-folder genomes/

ncbi-datasets CLI

Modern NCBI tool to download genomes, metadata, annotations

Advantages over older tools (JSON output, batch downloads)

Example: datasets download genome taxon “Escherichia coli” –reference –dehydrated

sra-tools (prefetch, fasterq-dump)

Download raw read data from SRA

prefetch to download .sra files

fasterq-dump to convert to FASTQ

Example workflow: prefetch SRR12345678 fasterq-dump SRR12345678
